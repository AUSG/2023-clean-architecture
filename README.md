# 2023 클린 아키텍쳐 스터디

클린아키텍쳐를 함께 읽자!

## 스터디 룰 
- 스터디는 매주 일요일 오전 10시 진행
  - 발표자는 당일 랜덤 2명 선정하여 진행
- 스터디 자료 등록 필수
  - PDF, 마크다운 어떤 형식이던 ok
- 질문 2개 이상 필수
  - 각자 궁금하거나 논의해보고 싶은 것을 사전 질문으로 2개 이상 남기기
